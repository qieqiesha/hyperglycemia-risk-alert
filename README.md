# hyperglycemia-risk-alert
本项目以高血糖（糖尿病前期/糖尿病） 为核心慢病对象，围绕临床需求构建兼具精度与可解释性的风险预警模型，实现高血糖风险预警。 项目基于公开健康数据集，整合年龄、BMI、血糖、血脂、家族史等多维度特征，完成数据整理与指标对齐，形成标准化建模样本；采用数据预处理流程，开展异常值核查、标准化与特征相关性分析，提升数据质量，为模型训练提供可靠支撑；使用Scikit-learn构建分类模型，结合SHAP可解释算法，量化指标贡献度，实现高血糖风险精准预测与个体归因解释；输出患者级风险解释案例，系统分析误判原因与使用注意事项，提升模型可信度，满足临床辅助决策需求。
